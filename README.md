# cap-n-snap
Cross-platform quick-share utility similar to puush or screencloud.

#Features:
- lol I take screenshots

# Prerequisites
- Python3
- Qt 5

Linux probably has these already.

# Python packages
Listed in requirements.txt

`sudo -H pip3 install -r requirements.txt`

# Running
`python3 main.py`
